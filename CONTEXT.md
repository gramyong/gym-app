# 운동 기록 웹앱

개인이 혼자 사용하는 모바일 우선 운동 기록 도구. 자유형 운동 기록과 통계 시각화를 제공한다.

## Language

**운동 세션 (Session)**:
사용자가 하루에 한 번 기록하는 단위. 날짜, 운동 이름, 소요 시간(분), 메모(선택)로 구성된다.
_Avoid_: 운동 기록, 로그, 워크아웃

**운동 이름 (Exercise)**:
세션에 연결된 운동의 명칭. 즐겨찾기에서 선택하거나 자유 입력으로 추가한다.
_Avoid_: 종목, 메뉴, 카테고리

**즐겨찾기 (Favorites)**:
사용자가 미리 등록해둔 자주 쓰는 운동 이름 목록. 세션 입력 시 탭으로 빠르게 선택한다.
_Avoid_: 태그, 프리셋, 템플릿

**소요 시간 (Duration)**:
세션의 운동 시간을 분(minutes) 단위로 기록한 값.
_Avoid_: 시간, 운동 시간, elapsed time

**통계 (Stats)**:
누적 세션 데이터를 집계한 뷰. 주간 막대 차트, 월간 히트맵, 추세선을 포함한다.
_Avoid_: 분석, 대시보드, 리포트

**히스토리 (History)**:
과거 세션 전체를 시간 역순으로 나열한 목록 뷰.
_Avoid_: 기록 목록, 로그, 피드

## Example Dialogue

> 개발자: "오늘 기록 추가할 때 운동 이름을 어떻게 입력해요?"
> 사용자: "즐겨찾기에 등록된 운동 목록이 뜨면 탭해서 선택하고, 없으면 직접 입력해요."
> 개발자: "즐겨찾기에 없는 걸 입력하면 자동으로 즐겨찾기에 추가되나요?"
> 사용자: "아니요, 즐겨찾기는 따로 관리해요. 세션은 그냥 저장되고요."
